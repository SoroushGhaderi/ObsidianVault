# Apache Kafka

Kafka is a distributed event streaming platform for building real-time data pipelines and streaming applications.

## Key Features
- **High throughput**: Millions of events per second.
- **Durability**: Persistent storage with replication.
- **Scalability**: Partitioned topics across clusters.
- **Real-time**: Low-latency streaming.

## Use in Chatbots
- Event-driven backend communication.
- Streaming AI responses token-by-token.
- Decoupling chat, AI, and analytics services.
- Audit logging and replay capability.
