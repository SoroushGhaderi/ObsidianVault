# Hubs

Topic cluster pages. The retrieval layer above the flat index.

**LLM:** create a hub when 5+ pages share a clear theme (e.g. "data-modeling", "airflow", "mental-models").
Hub entry format: `[[page-slug]] — one sentence`

**Query protocol when index > 60 entries:**
1. Scan index titles only
2. Identify relevant hub
3. Read hub → read specific pages
Do not read the full index. Hubs are your context budget.
