- [AI Chatbot Architecture: A Complete Overview 2026](https://www.nadcab.com/blog/ai-chatbot-architecture-guide)
  Use for: Microservices patterns, event-driven queues, scalability design.
- [Event-Driven Architecture for Chatbot APIs](https://chatspark.io/blog/event-driven-architecture-for-chatbot-apis)

- [eevee.bot architecture](https://eevee.bot/docs/design/architecture/)
  Use for: Microservices chatbot framework on Kubernetes, NATS messaging.
- [System Design for Chatbot - InterviewBit](https://www.interviewbit.com/blog/system-design-for-chatbot/)
  Use for: High-level system design patterns for interviews.

  Use for: Retrieval-Augmented Generation patterns and implementation.
- [Kafka Documentation](https://kafka.apache.org/documentation/)
  Use for: Event streaming platform for scalable chatbot backends.
- [PostgreSQL Documentation](https://www.postgresql.org/docs/)
  Use for: Database design for chat history and knowledge storage.
