# Lesson 1: Four Information Zones

Goal: classify information correctly before asking an LLM to write.

- `raw/` preserves original evidence.
- `wiki/` stores reusable synthesis.
- `journal/` captures the human activity stream.
- `log/` records system operations.

## Exercise

Classify an original article, a daily debugging observation, a durable technical concept, an ingest record, and an accepted architectural choice.

Answer: `raw/`, `journal/`, `wiki/concepts/`, `log/`, and `wiki/decisions.md`, respectively.
